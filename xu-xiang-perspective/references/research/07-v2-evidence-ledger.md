# 07 投资人 V2 证据账本

## 迁移结论

- 迁移等级：M2 增量补研。
- 公开证据等级：B 研究视角。
- 原因：本人系统表达极少，合法交易部分主要依赖一次完整采访、监管或法院材料和同期媒体复原。
- 不能确认：完整组合、固定仓位比例、产品级风险预算、精确成本、统一目标价和出狱后的新方法。

## 证据标签

| 等级 | 本 Skill 中的含义 |
|---|---|
| `[P1]` | 徐翔本人可核验完整采访 |
| `[P2]` | 法院、行业协会、公司公告或产品公开材料 |
| `[S1]` | 同期权威媒体对交易、组织和案例的复原 |
| `[S2]` | 后续评论或非同期转述 |
| `[I]` | 基于公开行为的框架推演 |
| `[U]` | 无法验证或来源冲突 |

## 核心命题审计

| 命题 | 证据 | 结论 |
|---|---|---|
| 徐翔偏好反向验证简单因果 | `[P1]` 澎湃面对面采访 | 可进入核心模型 |
| 长期高强度盯盘、研究和复盘 | `[S1]` 人民网等人物报道 | 可作为外部观察，不写成本人方法论 |
| 早期模式重视强势、成交和快速退出 | `[S1]` 同期营业部与人物报道 | 可用于公开短线框架，具体规则未知 |
| 规模扩大后转向机构化研究 | `[P2/S1]` 泽熙设立、产品与团队报道 | 可进入容量适配模型 |
| 重庆啤酒属于极端情绪后的反向交易 | `[S1]` 同期交易复原 | 可作为案例，不推广为通用抄底公式 |
| 历史收益可全部归因于合法能力 | `[P2]` 法院与中基协材料反证 | 明确拒绝 |
| 出狱后形成了新的系统方法 | `[U]` 无可信完整访谈 | 不采用 |

## 使用限制

- 研究来源数量未达到投资人 V2 的 A 级门槛。
- 法院和监管材料可以证明违法行为及边界，不能自动复原合法交易动机。
- 媒体报道可以描述行为，但不能替代本人对仓位、估值和退出规则的解释。
- 所有当前行情应用必须重新获取最新数据。
