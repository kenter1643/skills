# 徐翔 · 公开行为交易框架

基于可核验公开采访、历史交易报道、法院判决和监管材料提炼的 A 股短线分析 Skill。
已迁移到 `investor-v2`，当前公开证据等级为 **B 研究视角**。

## 适用场景

- 复盘公开市场中的强势股和催化剂
- 评估筹码结构、流动性和风险收益
- 为短线候选设置触发、失效、退出和仓位条件
- 区分可合法复用的公开市场方法与违法行为
- 复原三个历史案例，并明确仓位、成本和目标价中的未知项

## 触发示例

```text
用徐翔的视角复盘
徐翔模式会怎么看这个短线机会
用总舵主视角评估强势股
分析这个机会的风险收益
给出触发条件和退出条件
```

## 安装

```bash
npx skills add kenter1643/skills --skill xu-xiang-perspective
```

技能定义见 [SKILL.md](SKILL.md)，完整研究材料见 `references/research/`。

## 边界

本 Skill 仅使用公开信息和合法交易方式，不提供内幕交易、市场操纵或其他违法方法，也不直接喊单或保证收益。
由于徐翔本人系统公开表达极少，本 Skill 不虚构固定仓位、精确成本、实时观点或出狱后的新方法。
