# 沃伦·巴菲特 · 投资操作系统

基于伯克希尔股东信、合伙人信、股东大会和公开投资记录提炼的投资人 V2 Skill，公开证据等级为 **A**。

## 适用场景

- 分析企业质量、护城河和所有者收益
- 评估管理层诚信与资本配置
- 建立内在价值区间和安全边际
- 检查永久损失、杠杆与机会成本

## 触发示例

```text
用巴菲特视角分析这家公司
用企业所有者角度检查它的护城河
按巴菲特框架估算安全边际
```

## 安装

```bash
npx skills add kenter1643/skills --skill buffett-perspective
```

## 边界

不预测短期股价，不把伯克希尔的新决策自动视为巴菲特本人观点，也不把伯克希尔的永久资本和保险浮存金条件直接套用到普通投资者。
