# 06 人物时间线

调研截止：2026-06-09

| 时间 | 事件 | 思想影响 |
|---|---|---|
| 1930 | 出生于美国奥马哈 | 对美国商业、储蓄和复利形成早期兴趣 |
| 1940s | 送报、经营弹球机等小生意 | 从单位经济和现金回报理解商业 |
| 1950-1951 | 就读哥伦比亚，师从Benjamin Graham | 建立价格与价值、安全边际框架 |
| 1956 | 创建Buffett Partnership | 运用低估证券、套利和控制型投资 |
| 1965 | 控制Berkshire Hathaway | 从证券组合转向永久资本和企业集团 |
| 1967 | 收购National Indemnity | 保险浮存金成为资本配置引擎 |
| 1972 | 收购See's Candies | 转向重视品牌、定价权和资本效率 |
| 1984 | 发表“Graham-and-Doddsville的超级投资者” | 系统捍卫价值投资与安全边际 |
| 1988 | 大举投资Coca-Cola | 优质生意、长期持有和集中投资成熟 |
| 1999 | 公开讨论市场长期回报与互联网泡沫 | 强化能力圈和反短期预测 |
| 2006 | 宣布逐步捐出绝大多数财富 | 财富托管与社会回馈进入核心叙事 |
| 2008 | 金融危机中提供资本 | 流动性、信誉和逆周期能力得到验证 |
| 2016 | 开始建立Apple持仓 | 能力圈从技术本身扩展到消费者生态 |
| 2023 | Charlie Munger去世 | 合伙关系与制度传承成为晚期主题 |
| 2025-05-03 | 宣布年末卸任CEO并建议Greg Abel接任 | 完成经营权交接 |
| 2025-11-10 | 发布Thanksgiving message | 宣布不再写年度股东信或主持长时间大会问答 |
| 2026-01-01 | Greg Abel成为Berkshire CEO | 巴菲特转为董事长、主要股东和顾问 |
| 2026 | 仍在办公室工作并参与资本配置讨论，但不再担任CEO | Skill对经营现状应以Greg Abel披露为准 |

## 最新状态

- 截至2026年6月，巴菲特为Berkshire Hathaway董事长，不再是CEO。
- 2025年度股东信由Greg Abel于2026年发布，这是接班后首封年度信。
- 巴菲特表示未来主要通过年度Thanksgiving message继续沟通。
- 其伯克希尔股份将持续捐赠，并计划在去世后约十年内完成剩余股份的公益分配。
