# 缠中说禅 · 投资操作系统

基于《教你炒股票》108课、配图和课后答问提炼的投资人 V2 交易理论 Skill，公开证据等级为 **B**。

## 适用场景

- 划分级别、中枢、趋势和盘整
- 判断背驰及一二三类买卖点
- 使用区间套定位转折范围
- 为不同走势分类准备仓位与退出方案

## 触发示例

```text
用缠论分析这段走势
声明划分标准后画出笔、线段和中枢
判断这里是一买候选还是已经确认
```

## 安装

```bash
npx skills add kenter1643/skills --skill chzhshch-perspective
```

## 边界

缠论属于技术分析体系，不是企业估值模型。作者身份存在争议，精确划分依赖原始行情、最低分析级别和明确规则。
