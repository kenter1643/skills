# 缠中说禅 · 走势结构操作系统

基于《教你炒股票》108课、配图及课后答问提炼的走势分析 Skill，严格区分原著定义和后世简化缠论。

## 适用场景

- 分析指数、股票、期货等连续交易品种
- 划分级别、中枢、背驰和一二三类买卖点
- 使用区间套定位，并为不同走势分类准备操作方案

## 触发示例

```text
用缠论分析这段走势
缠中说禅怎么看这只股票
画出笔、线段和中枢
判断这里是不是一买或二买
用区间套定位买卖点
```
