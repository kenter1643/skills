# 05 实盘与操作决策

## 可提炼的操作模式

### 1. 先固定操作级别

- 时间、资金、反应速度不同，选择的最低分析级别不同。
- 技术不熟或无法盯盘者应放大级别，避免一分钟级别频繁操作。

### 2. 只在结构确认后行动

- 第一类买卖点来自趋势背驰。
- 第二类买卖点来自第一类点后的首次次级别回抽。
- 第三类买卖点来自离开中枢后回抽不进入中枢。

### 3. 区间套缩小定位

- 从大级别背驰段逐级进入次级别，寻找更精确的转折结构。
- 不能直接用一个MACD金叉或单根K线替代完整结构。

### 4. 同级别分解

- 将走势按统一规则分解，减少持仓中随意切换观察级别。
- 小级别介入就按小级别退出，大级别介入则容忍小级别波动。

### 5. 风险与仓位

- 学习阶段不应全仓练习。
- 卖点应在上涨中处理，不能恐慌追杀。
- 赚钱卖出不必后悔；卖错后等待新买点，不情绪化追回。
