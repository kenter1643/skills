# 林广茂 · 投资操作系统

基于林广茂公开演讲、郑商所论坛问答、监管文件和公开投资行为提炼的投资人 V2 Skill，公开证据等级为 **B**。

## 适用场景

- 棉花及其他农产品供需分析
- 现货、基差、仓单和交割逻辑
- 大级别趋势识别与持有
- 判断交易方法是否适合当前市场和资金规模
- 分析策略优势是否已经衰减
