# 06 人物时间线

| 时间 | 事件 | 对框架的影响 |
|---|---|---|
| 1877 | 出生于马萨诸塞州 | 历史人物，材料不存在“近期动态” |
| 约1891 | 成为报价板记录员 | 形成价格记忆和先例观察 |
| 1890年代 | 在 bucket shops 交易 | 擅长短期波动与快速反馈 |
| 约1900 | 进入纽约真实交易所环境 | 原策略因执行机制变化失效 |
| 1906-1907 | 在剧烈下跌与恐慌中建立声名 | 强化主趋势和阻力最小方向 |
| 1908前后 | 棉花交易重创 | 听取意见、违背系统的代表案例 |
| 1915 | 破产阶段 | 重新认识等待和机会选择 |
| 1922-1923 | Lefevre 连载并出版《Reminiscences》 | 形成最广泛流传的利弗莫尔形象 |
| 1924 | 同时代报刊报道其系统与经历 | 提供文学作品之外的交叉材料 |
| 1929 | 因做空崩盘而成为传奇人物 | 具体利润数字存在版本差异 |
| 1934 | 再次破产 | 暴露杠杆与财富保护缺陷 |
| 1940-03 | 出版《How to Trade in Stocks》 | 本人署名的系统性总结 |
| 1940-11-28 | 在纽约去世 | 其人生结局构成重要风险边界 |

## 史料说明

- 生卒和出版时间可由图书馆目录、同期报刊交叉确认。
- 具体交易利润、破产次数、J.P. Morgan 或总统亲自要求其停止交易等故事，版本冲突较多。
- 不以 Wikipedia 或后世励志文章作为核心模型证据。
