# 杰西·利弗莫尔 · 投资操作系统

基于杰西·利弗莫尔的《How to Trade in Stocks》、Edwin Lefevre 的文学化访谈作品
《Reminiscences of a Stock Operator》、历史报刊和现代注释研究提炼的投资人 V2 Skill，公开证据等级为 **B**。

## 适用场景

- 股票、期货和指数的趋势判断
- 突破确认、试探仓与顺势加仓
- 交易频率、持仓耐心和退出纪律
- 区分正常回撤与趋势失效
- 复盘“判断正确但为什么没有赚到钱”

## 触发示例

```text
用利弗莫尔的视角分析这个品种
利弗莫尔会怎么看这次突破
判断现在是不是阻力最小方向
这笔仓位应该持有、加仓还是退出
用股票作手框架复盘我的交易
```

## 边界

本 Skill 不代表利弗莫尔本人，不把文学化叙事当逐字口述，不复刻其高杠杆、坐庄或市场操纵。
涉及现实行情时必须获取最新数据，并给出触发、失效、仓位和退出条件。

技能定义见 [SKILL.md](SKILL.md)，研究材料见 `references/research/`。
