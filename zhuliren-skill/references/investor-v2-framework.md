# 投资人 V2：研究与提炼协议

当目标人物的主要专业能力来自投资、交易、资产配置或资本配置时，使用本协议替代通用人物的六维研究流程。

## 1. 目标

投资人 V2 不以“像本人说话”为主要目标，而是复原一套可审计的投资操作系统：

```text
优势来源 -> 研究流程 -> 估值与预期 -> 组合与仓位 -> 买卖纪律 -> 证伪与复盘
```

所有结论必须区分：

- 本人明确表达
- 本人可核验行为
- 基于框架的推演
- 无法验证的传闻

## 2. 证据等级

| 标签 | 类型 | 可进入核心模型 |
|---|---|---|
| `[P1]` | 本人著作、股东信、基金信、演讲原文、监管原件 | 是 |
| `[P2]` | 完整访谈、公司官方材料、公开持仓或交易披露 | 是，但行为不能自动等同于动机 |
| `[S1]` | 权威传记、同期权威报道、法院或交易所整理材料 | 可辅助 |
| `[S2]` | 后世研究、评论与非同期转述 | 仅作补充 |
| `[I]` | 根据公开行为作出的推断 | 只能标注后使用 |
| `[U]` | 无法验证、来源冲突或网络传闻 | 否 |

核心模型至少需要两条相互独立的 `[P1]`/`[P2]` 证据，或一条 `[P1]` 加一条高可信行为证据。

### 01 一手来源与证据账本

- 建立来源清单、日期、URL、证据等级和可支持的命题。
- 区分“说过”“做过”“别人说他做过”。
- 记录冲突、缺页、转述链和不可访问原件。

### 02 投资哲学与优势来源

- 市场为何错误定价。
- 人物依靠什么获得优势。
- 风险的定义、能力圈和明确放弃的机会。
- 收益主要来自价值重估、成长、趋势、事件、套利、杠杆还是流动性溢价。

### 03 研究系统

- 研究从哪里开始，先排除什么。
- 第一信息源、关键指标和交叉验证方式。
- 公司、产业、宏观、价格行为分别占什么位置。
- 什么条件下停止研究或承认无法判断。

### 04 估值与预期

- 使用的估值锚、情景和基准率。
- 如何处理增长、周期、终值和不可估值资产。
- 当前价格隐含什么预期。
- 安全边际或赔率如何形成。

### 05 组合与仓位

- 初始仓位、最大仓位、加仓条件和集中度。
- 现金、杠杆、相关性、流动性和风险预算。
- 组合层面最大损失或不可承受风险。
- 无公开证据的具体比例必须写“未知”。

### 06 交易与退出

- 观察、建仓、确认、加仓、减仓和退出。
- 价格止损、逻辑止损、时间止损和对冲。
- 哪些情况下价格波动不是退出理由。
- 极端行情和流动性枯竭如何处理。

### 07 案例与决策

至少复原三个完整案例：

```text
当时可得信息 -> 原始判断 -> 估值/赔率 -> 仓位或行动 -> 后续变化 -> 结果 -> 可复制部分
```

禁止用事后结果倒推人物当时一定知道什么。

### 08 失败、争议与矛盾

至少两个失败或争议案例，并区分：

- 判断错误
- 估值错误
- 时机错误
- 仓位错误
- 流动性或杠杆错误
- 纪律、组织或合规错误

### 09 表达DNA

表达风格只负责呈现，不得改变事实置信度，也不得把框架推演伪装成本人原话。

### 10 时间线与市场环境

- 投资生涯关键节点和观点演化。
- 方法在哪些市场环境有效或失效。
- 活人必须覆盖最近12个月的公开变化。

## 4. 提炼要求

最终 Skill 必须明确：

1. 适用市场、资产和持有周期。
2. 核心收益来源及其持续条件。
3. 研究流程和信息优先级。
4. 估值、市场预期和催化剂。
5. 组合、仓位、相关性和流动性。
6. 建仓、加仓、减仓和退出规则。
7. 至少三条证伪或失效条件。
8. 方法适用及不适用的市场环境。
9. 成功案例、失败案例和不可复制部分。
10. 合规边界及无法确认的信息。

## 5. 交付等级

| 等级 | 条件 | 可用范围 |
|---|---|---|
| A 可实战调用 | 来源、案例、仓位、退出、证伪和环境适配均通过 | 可分析具体标的，但仍需最新数据 |
| B 研究视角 | 哲学和研究系统可靠，执行或仓位证据不足 | 可做研究框架，不给具体仓位 |
| C 人物理解 | 主要是二手材料或表达样本 | 不用于投资决策 |

等级不是人物能力评价，而是公开证据对 Skill 的支持程度。
