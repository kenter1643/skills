# 李录 · 投资操作系统

基于李录公开演讲、著作、Himalaya官方材料和可核验披露提炼的投资人 V2 Skill，公开证据等级为 **A**。

## 适用场景

- 深度研究企业历史、竞争格局和管理层
- 检查知识边界、反证和安全边际
- 分析长期复利、集中投资和资本期限
- 评估中国与全球现代化背景下的企业机会

## 触发示例

```text
用李录视角分析这家公司
按知识诚实框架检查我的投资逻辑
用喜马拉雅资本视角做反方研究
```

## 安装

```bash
npx skills add kenter1643/skills --skill li-lu-perspective
```

## 边界

13F只反映部分美国证券的滞后披露，不代表完整组合、实时仓位或李录个人观点。
