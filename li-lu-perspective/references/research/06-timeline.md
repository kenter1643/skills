# 06 人物时间线

调研截止：2026-06-09

| 时间 | 事件 | 思维影响 |
|---|---|---|
| 1966 | 出生于中国河北唐山 | 早年经历塑造对历史、制度和生存的关注 |
| 1985 | 进入南京大学学习 | 接触现代知识与公共事务 |
| 1989 | 参与中国学生运动，后离开中国 | 个人自由、责任和制度成为长期主题 |
| 1990s | 就读哥伦比亚大学，同时获得BA、JD、MBA | 法律、商业和跨学科训练 |
| 1993前后 | 听到Warren Buffett在哥伦比亚演讲 | 转向价值投资 |
| 1997 | 创立Himalaya Capital | 开始长期投资实践 |
| 2000s | 与Charlie Munger建立长期师友和投资关系 | 强化集中、品格、跨学科与长期复利 |
| 2000s | 投资BYD并介绍给Munger | 企业家、工程文化和长期成长案例 |
| 2010 | 为中文版《穷查理宝典》作序 | 系统传播Munger思想 |
| 2013 | 接受Graham & Doddsville访谈 | 系统表达知识诚实与完整信息 |
| 2015 | 北大演讲“中国价值投资的展望” | 连接价值投资与中国现代化 |
| 2017-2024 | 担任Columbia University trustee | 教育与机构受托实践 |
| 2018 | 成为Caltech trustee | 科学、教育和长期机构治理 |
| 2020 | 出版《文明、现代化、价值投资与中国》 | 汇总文明尺度的投资框架 |
| 2020-2024 | The Asian American Foundation首任董事长 | 公益、社会信任和责任 |
| 2023 | 获Aspen Institute Henry Crown Leadership Award | 领导与公共贡献获认可 |
| 2024-12 | 北大发表“全球价值投资与时代” | 最新系统性全球价值投资表达 |
| 2026-05 | Himalaya提交最新13F | 公开组合维持集中，但信息仅为局部快照 |
