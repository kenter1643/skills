## 工作流程（v3.4 全维度版）

### Step 0: 宏观情绪预判（新增前置步骤）

在获取涨停数据前，先扫描外部环境，综合打出 **-5 到 +5** 的情绪分。

#### 0.1 海外股市收盘（权重 1.5）
| 条件 | 得分 |
|------|:--:|
| 美股三大指数全红 + 费城半导体涨 >1% | +1.5 |
| 涨跌互现 | 0 |
| 三大指数全绿或费城半导体跌 >1% | -1.5 |

#### 0.2 比特币（权重 1.5）
| 条件 | 得分 |
|------|:--:|
| BTC 日涨 >2% 且恐惧贪婪指数 >50 | +1.5 |
| BTC 震荡 (±2%内) | 0 |
| BTC 日跌 >3% 或恐惧贪婪指数 <20 | -1.5 |

#### 0.3 汇率 + 美债（权重 1.0）
| 条件 | 得分 |
|------|:--:|
| USDCNH 走弱(人民币升) + 美10Y稳 | +1.0 |
| 稳定 | 0 |
| USDCNH 破 7.30 或美10Y 破 4.5% | -1.0 |

#### 0.4 A50期货 + 盘前政策（权重 1.0）
| 条件 | 得分 |
|------|:--:|
| A50 涨 + 有明确利好政策 | +1.0 |
| 平 | 0 |
| A50 跌或重大利空 | -1.0 |

#### 0.5 情绪分判定

| 总分 | 判定 | 对后续步骤的影响 |
|:----:|------|------------------|
| ≥ 3 | 偏乐观 | 正常执行，可推荐 3 只 |
| 1 ~ 2 | 中性 | 正常执行，但谨慎推荐最多 2 只 |
| 0 | 偏空 | 自动降级为「谨慎推荐」，最多 1 只 |
| ≤ -1 | 明确偏空 | 无论后续过滤结果如何，**仅允许灰色试探或禁止** |

### Step 1: 获取当日涨停复盘数据

通过网络搜索获取当日A股涨停复盘核心数据：
- **来源优先级**：财联社焦点复盘 > 证券时报 > 韭研公社
- **核心字段**：涨停家数（不含ST）、封板率、连板晋级率、炸板数
- **辅助字段**：大盘涨跌、成交额、涨跌家数比、连板梯队、首板板块分布
- **口径要求**：统一使用**不含ST股**口径；若数据源口径冲突，以财联社为准

### Step 2: v3.4 环境过滤（六关逐级判断）

```
第一关 → 硬性禁止（同 v3.2）
├─ 涨停 ≥ 100       → 禁止（高潮日，1进2成功率极低）
├─ 涨停 < 40        → 禁止（流动性枯竭）
├─ 封板率 < 65%     → 禁止（炸板率高，情绪差）
└─ 触发任一 → 输出「今日禁止推荐」+ 原因 + 市场阶段判断，终止

第二关 → 降级检查（v3.2）
└─ 前一日 (涨停 ≥ 80 且 封板率 > 85%)？
    ├─ 是 → 降级为「谨慎推荐」（最多推荐 1-2 只，仓位减半）
    └─ 否 → 继续

第三关 → 板块拥挤度预警（新增 v3.4）
└─ 检查当日涨停板块分布
    └─ 单一板块涨停占全市场 > 25%？
        ├─ 是 → 该板块 ⚠️ ，所有内部首板自动降一级评级
        └─ 否 → 无影响

第四关 → 主线过热拦截（新增 v3.4）
└─ 同一主线连续领涨天数
    ├─ ≥ 3 天连续领涨 → ⚠️ 主线过热，仅推荐非主线方向首板
    ├─ 2 天连续领涨 → 标注 [⚠ 主线可能次日分化]
    └─ ≤ 1 天 → 无影响

第五关 → 必须满足（同 v3.2）
├─ 50 ≤ 涨停 < 100 且 封板率 ≥ 65% 且有明确上升主线
│   └─ 满足 → 正常推荐（最多 3 只，受 Step 0 情绪分约束）
└─ 不满足但 40 ≤ 涨停 < 50 → 灰色试探（最多 1 只，极轻仓位）

第六关 → 辅助过滤（同 v3.2）
└─ 前一日 1进2 晋级率 < 10%？
    ├─ 是 → 标注 [⚠ 前日极低晋级率]
    └─ 否 → 无标注
```

### Step 3: 首板筛选（五维度评分）

| 维度 | 权重 | 评分标准 |
|------|:----:|----------|
| 板块强度 | 30% | 当日板块涨停家数排名，优先 Top3；受拥挤度降级的板块权重折半 |
| 封板质量 | 25% | 封单金额、封板时间（越早越好）、有无盘中炸板、最终封板 |
| 题材催化 | 20% | 政策/产业/事件催化，区分一次性消息 vs 有持续性的产业趋势 |
| 辨识度 | 15% | 流通市值适中（50-200亿最优）、历史股性活跃、形态位置合理 |
| 梯队完整 | 10% | 所属板块连板梯队是否完整（有3连板以上龙头带队加分） |

### Step 3.5: 个股基本面过滤（新增 v3.4）

对五维度评分后的 Top 候选股逐只审查：

#### 亏损股识别（自动降为 C 级）
满足以下任一项：
- 最近季报/年报扣非净利润 < 0
- 连续两个季度营收萎缩
- 经营现金流持续为负

#### 纯消息股识别（满足 ≥3 项 → 自动降为 C 级）
| 特征 | 标准 |
|------|------|
| 扣非负 | 主营业务亏损 |
| 无机构买入 | 龙虎榜近1月无机构席位净买入记录 |
| 游资占比高 | 龙虎榜当日游资买入占比 > 15% |
| 已发澄清公告 | 公司就传闻发布澄清/风险提示公告 |

#### 龙虎榜质量加分
| 条件 | 加分 |
|------|:--:|
| 机构席位净买入 | +2 |
| 知名游资参与（方新侠/章盟主/炒股养家等） | +1 |
| 上榜但无机构/游资 | 0 |
| 未上榜 | -1（透明度不足） |

#### 基本面标记
| 标记 | 含义 | 处理 |
|:--:|------|------|
| 🟢 | 扣非正 + 有机构 + 无澄清公告 | 正常 |
| ⚠️ | 有争议但未触发降级 | 标注风险 |
| 🔴 | 亏损股或纯消息股 | 自动降 C 级，不推荐 |

### Step 4: 综合定级与输出

结合技术面评分 + 基本面标记，生成最终评级：

| 最终级别 | 条件 | 含义 |
|----------|------|------|
| S 级 | 五维评分 Top + 🟢 + 板块健康 + 龙头带队 | 高概率晋级，首选 |
| A 级 | 五维评分较高 + 🟢 | 较好概率 |
| B 级 | 五维评分中等或 ⚠️ | 中等概率，需盘中确认 |
| C 级 | 🔴 或五维评分低 | 不推荐参与 |

按模板输出。

## 核心信条

1. **空仓是能力**：80% 时间不交易，只在 20% 高胜率窗口出手
2. **封单大≠次日接力意愿**：封单是滞后指标，涨停次日才是真正的博弈
3. **板块涨停最多≠个股安全**：拥挤板块次日分化最惨烈（→ v3.4 25%阈值）
4. **高潮日不追、过热主线不碰**：涨停≥100 或主线连涨≥3天 → 次日是陷阱
5. **首板晋级率与涨停总数成反比**：涨停越多，首板溢价越稀薄
6. **消息面是安全垫，不是冲锋号**：利好消息+亏损股 = 陷阱；利好+业绩股 = 机会
7. **比特币领先 0.5~1 天**：BTC跌幅 >3% = A股次日大概率承压，主动降仓

## 边界情况

| 场景 | 处理 |
|------|------|
| 当日非交易日 | 提示"今日非交易日，请在工作日 A股交易时段使用" |
| 多板块并列最强 | 优先选梯队完整且有龙头带队的板块 |
| 板块同时触发拥挤+过热 | 该板块首板全部排除，优先其他方向 |
| 所有 Top 候选均为 🔴 | 输出"今日无符合条件的标的，建议空仓" |
| 个股同时多概念 | 以当日最强概念为第一标签 |
| 首板次日一字涨停 | 标注"可能无参与机会" |
| 搜索数据不完整 | 标注数据缺失项，降低推荐评级的置信度 |
| 情绪分与过滤结果矛盾 | 以更保守的一方为准（如情绪-1 但技术面 OK → 灰色试探） |
