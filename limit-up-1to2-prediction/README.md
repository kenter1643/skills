# limit-up-1to2-prediction

A股短线首板晋级二板预测技能 v3.5，整合宏观情绪预判 + 结构性高低切复核 + 板块拥挤度预警 + 主线过热拦截 + 个股基本面过滤 + 技术面五维度评分。

## 核心能力

| 步骤 | 内容 | 维度 |
|------|------|------|
| Step 0 | 宏观情绪预判 | 美股/BTC/汇率+美债/A50+政策 → -5 ~ +5 |
| Step 1 | 涨停复盘数据获取 | 财联社/证券时报/韭研公社 |
| Step 2 | 六关逐级过滤 | 硬性禁止→结构性高低切复核→降级→板块拥挤度>25%→主线过热≥3天→必须满足→辅助过滤 |
| Step 2.5 | 盘前二次复核 | 公告/监管/澄清/隔夜风险 → 保留、降级或剔除 |
| Step 3 | 五维度评分首板 | 板块强度(25%)+封板质量(25%)+题材催化(20%)+辨识度(25%)+梯队(5%) |
| Step 3.5 | 个股基本面过滤 | 亏损股/纯消息股自动降C + 龙虎榜加分 |
| Step 4 | 综合定级输出 | S/A/B/C + 消息面标记 + 情绪分行 |

## v3.5 核心修正

### 涨停过百不再一刀切

当涨停家数 ≥100 时，先做「结构性高低切复核」：

| 条件 | 标准 |
|------|------|
| 指数弱、个股强 | 指数收绿或分化，但上涨家数/涨停家数明显扩张 |
| 资金切换明确 | 高位旧主线退潮，低位新分支承接 |
| 新分支有梯队 | 新方向出现2板以上龙头或容量中军共振 |
| 封板率健康 | 封板率 ≥70% |
| 次日催化延续 | 盘后/隔夜有持续性催化 |

- 复核分 ≥3：认定为结构性高低切，降级为「谨慎推荐」
- 复核分 1-2：仅列「观察池」，不做正式推荐
- 复核分 0：普涨高潮日，禁止推荐

### 提高辨识度权重

回测显示，春秋电子、中京电子、达实智能这类人气核心/容量核心更容易穿越分歧。v3.5 将辨识度权重从 15% 提高到 25%，并将 20cm/30cm 弹性首板默认降为情绪指标，除非开盘承接和板块共振继续超预期。

### 新增盘前二次复核

次日开盘前必须检查公告、监管、澄清和隔夜风险。若公司澄清不涉及热门概念、遭遇关注函或 BTC/外盘科技线明显走弱，候选股自动降级或剔除；若利空后竞价仍明显强于板块，则保留并标注「资金无视利空」。

## 核心信条

1. **空仓是能力**：80% 时间不交易，只在 20% 高胜率窗口出手
2. **封单大≠次日接力意愿**：封单是滞后指标，涨停次日才是真正的博弈
3. **板块涨停最多≠个股安全**：拥挤板块次日分化最惨烈（→ 25% 阈值）
4. **高潮日先复核结构**：涨停≥100 不是一刀切，若是旧主线退潮后的低位新主线接力，可降级观察
5. **首板晋级率与涨停总数通常成反比**：涨停越多，首板溢价越稀薄；但结构性高低切日优先看辨识度核心
6. **消息面是安全垫，不是冲锋号**：利好消息+亏损股 = 陷阱；利好+业绩股 = 机会
7. **比特币领先 0.5~1 天**：BTC 跌幅 >3% = A股次日大概率承压
