# Financial market analysis framework

当用户需要以下任务时使用此 skill：

- A股盘前分析
- 港股盘前分析
- 全球市场周度展望
- 重大事件周的资产联动分析
- 将一段市场分析内容提炼成可复用的方法论或提示词

## 用户固定偏好

在做 A股 / 港股盘前或盘中判断时，默认把“昨晚美股走势及其驱动因素”作为判断因子之一，重点关注：

- 标普 500、纳斯达克、道琼斯强弱
- 科技 / 金融 / 能源板块表现
- 美元指数变化
- 原油与黄金变化
- 地缘风险、政策冲击、风险偏好传导

## 核心方法论

整体按以下顺序组织：

1. 先识别当前市场的第一核心变量
2. 先做全球资产扫描，再做分市场判断
3. 各资产尽量统一按以下结构输出：
   - 核心判断
   - 驱动逻辑
   - 风险点 / 失效条件
   - 关键价位 / 预估区间
   - 交易或配置启示
4. 必须做情景推演，避免单边拍脑袋式预测
5. 最终结论必须落到仓位、节奏、方向和风险控制

## 推荐输出结构

### 1. 总论
- 当前最大变量是什么
- 该变量为何决定未来 1 天 / 1 周风险偏好
- 当前属于趋势延续、情绪修复还是事件扰动

### 2. 全球市场温度计
默认覆盖：
- 美股三大指数
- 美元指数
- 美债收益率（如有）
- 人民币汇率
- 原油
- 黄金
- 比特币
- A股主要指数
- 港股主要指数

### 3. 分市场分析
至少评估：
- 美元与人民币汇率
- 美股
- 港股
- A股
- 原油 / 黄金 / 大宗商品
- 比特币或主要加密资产（如相关）

### 4. A股 / 港股结构判断
重点判断：
- 普涨、轮动还是分化
- 高低切换还是继续抱团
- 主线、补涨、情绪博弈方向分别是什么

### 5. 情景推演
至少给出：
- 情景 A：利多风险资产
- 情景 B：利空风险资产
- 必要时加情景 C：中性震荡

### 6. 关键事件清单
按重要性排序，列出未来将影响市场波动的关键日期与事件。

### 7. 综合策略建议
必须落到：
- 仓位建议
- 节奏建议
- 可关注方向
- 应规避方向
- 哪些信号出现后需要修正判断

### 8. 风险提示
必须写清：
- 哪些变量尚不确定
- 是否存在 FOMO、超买、缩量上涨、技术背离等风险
- 仅供参考，不构成投资建议

## 表达要求

- 先结论，后论证
- 多用条件判断，少用绝对化措辞
- 尽量给区间、关键位、支撑阻力
- 不只复述新闻，要解释跨资产传导链条
- 不只判断指数，要判断行业主线、资金风格和高低切换
