# 金融市场盘前 / 周度分析提示词模板

## 使用说明

这是一份可长期复用的提示词模板，适用于：
- A股盘前分析
- 港股盘前分析
- 周度市场展望
- 重大事件周的多资产联动分析
- 需要把宏观、指数、行业、仓位建议放在一起输出的场景

核心要求：
1. 先抓主导市场的核心变量，不要一开始分散罗列信息。
2. 必须把昨晚美股走势及其驱动因素纳入 A股/港股判断因子，重点关注指数强弱、科技/金融/能源板块表现，以及油价、地缘风险等外部冲击。
3. 分析必须体现跨资产联动，不得孤立看单一市场。
4. 每个资产/市场都尽量按“核心判断 - 驱动逻辑 - 风险点 - 区间预判/关键位”展开。
5. 必须做情景推演，并把结论落到仓位和配置建议。
6. 必须附风险提示，不得写成确定性投资建议。

---

## 模板一：通用版提示词

请你扮演一名具备宏观视角、事件驱动分析能力、跨资产联动意识的专业金融市场分析师。围绕【{分析日期}】的市场环境，输出一份适用于【A股 / 港股 / 全球市场 / 周度展望】的分析报告。

请严格按照以下要求执行：

### 一、分析原则
1. 先识别当前决定市场方向的“第一核心变量”，例如：重大地缘事件、政策变化、通胀数据、财报季、美联储路径、汇率波动、商品价格冲击等。
2. 先做全球资产状态扫描，再展开分市场分析。
3. 重点纳入昨晚美股走势及驱动因素，尤其是：
   - 标普500、纳斯达克、道琼斯表现
   - 美股科技、金融、能源等关键板块强弱
   - 美元指数变化
   - 美债收益率变化（如有）
   - 原油、黄金走势
   - 地缘风险或政策扰动
4. 对 A股、港股分析时，要说明海外市场如何通过风险偏好、汇率、行业映射、外资预期影响本地市场。
5. 不要只给单边结论，必须进行情景推演。
6. 最终结论必须落到仓位、节奏、方向和风险控制建议。

### 二、输出结构
请按以下结构输出：

#### 1. 总论：当前市场最大变量
说明：
- 当前市场最重要的核心变量是什么
- 这一变量为何决定未来 1 天 / 1 周的市场风险偏好
- 当前属于趋势延续、情绪修复，还是事件扰动主导

#### 2. 全球市场温度计
请用简洁清晰的方式梳理以下资产当前状态：
- 美股三大指数
- 美元指数
- 美债收益率（如有）
- 人民币汇率
- 原油
- 黄金
- 比特币
- A股主要指数
- 港股主要指数

要求：
- 不只是罗列数据，还要概括风险偏好状态
- 说明哪些资产在相互验证，哪些资产在背离

#### 3. 分市场展开分析
请分别分析：
- 美元与人民币汇率
- 美股
- 港股
- A股
- 原油 / 黄金 / 大宗商品
- 比特币或主要加密资产（如相关）

每一部分尽量统一使用以下结构：
- 核心判断
- 驱动逻辑
- 利多因素
- 利空因素
- 风险点 / 失效条件
- 关键价位 / 震荡区间
- 对交易或配置的启示

#### 4. A股 / 港股结构主线判断
这里必须重点写：
- 当前市场是普涨、轮动还是分化
- 资金在高低切换还是继续抱团
- 哪些方向属于主线
- 哪些方向属于补涨
- 哪些方向属于风险较大的情绪博弈

如涉及 A股，请优先判断：
- 算力 / AI / 半导体
- 锂电 / 新能源
- 券商 / 保险 / 银行等金融板块
- 周期 / 资源 / 化工 / 有色
- 防御性方向

#### 5. 情景推演
至少给出两个情景：

情景 A：利多风险资产
- 触发条件
- 美股表现倾向
- 美元 / 原油 / 黄金变化方向
- A股 / 港股受益方向
- 应对策略

情景 B：利空风险资产
- 触发条件
- 美股表现倾向
- 美元 / 原油 / 黄金变化方向
- A股 / 港股承压方向
- 应对策略

如有必要，可增加情景 C：中性震荡。

#### 6. 关键事件日历
列出未来最关键的时间点：
- 重要政策
- 财报
- 宏观数据
- 地缘事件
- 监管表决
- 央行操作

并说明：
- 哪些事件最可能引发市场波动
- 哪些事件对哪类资产影响最大

#### 7. 综合策略建议
从实战角度给出：
- 当前仓位建议（偏进攻 / 中性 / 偏防守）
- 更适合追高还是等回调
- 可以关注的方向
- 应规避的方向
- 哪些市场信号一旦出现，需要调整原判断

#### 8. 风险提示
必须明确写出：
- 哪些变量尚不确定
- 当前是否存在 FOMO、超买、缩量上涨、技术性背离等风险
- 以上分析仅供参考，不构成投资建议

### 三、表达要求
1. 先结论，后论证。
2. 尽量给出区间、关键位、支撑阻力，不要只说“看多/看空”。
3. 多用条件判断，少用绝对化措辞。
4. 语言专业、清晰、可执行，避免空泛表述。
5. 如果缺乏实时数据，请明确标注“以下为基于已知信息的框架性推演”。

## 模板二：A股 / 港股盘前精简提示词

请基于最新可得市场信息，输出一份【今日 A股 / 港股盘前分析】。

请特别注意：
- 必须先总结昨晚美股表现及驱动因素
- 必须分析美元、人民币、原油、黄金对今日 A股 / 港股风险偏好的影响
- 必须判断今天市场更可能是普涨、分化、轮动还是冲高回落
- 必须给出行业主线、风险点和盘中重点观察信号

输出结构如下：
1. 一句话总判断
2. 昨晚美股与全球资产信号
3. 今日 A股 / 港股可能的情绪与指数节奏
4. 重点方向（按优先级排序）
5. 风险方向 / 回避方向
6. 今日关键观察指标
7. 仓位建议
8. 风险提示

如果遇到重大事件窗口，请额外加入“情景推演”。

## 模板三：周度展望加强版提示词

请围绕【{起始日期} - {结束日期}】输出一份“全球金融市场周度展望”。

要求：
1. 先识别未来一周最大的主导变量。
2. 从全球资产联动角度分析美元、美股、A股、港股、原油、黄金、比特币之间的传导关系。
3. 对每个市场都给出：方向判断、驱动逻辑、风险点、预估区间。
4. 必须给出未来一周最重要的事件清单，并按重要性排序。
5. 必须给出不同情景下的配置建议。
6. 必须单独写出“本周最值得警惕的风险信号”。

建议输出结构：
- 本周核心结论
- 本周最大变量
- 全球市场温度计
- 分资产判断
- A股 / 港股结构机会
- 本周关键事件清单
- 情景推演
- 综合策略建议
- 风险提示

## 模板四：可直接复制使用的极简版提示词

请按“核心变量 -> 全球资产扫描 -> 分资产分析 -> A股/港股结构判断 -> 情景推演 -> 关键事件 -> 仓位与配置建议 -> 风险提示”的框架，写一份专业的金融市场分析。

特别要求：
- 必须纳入昨晚美股走势及其驱动因素
- 重点关注科技、金融、能源板块表现
- 重点分析美元、人民币、原油、黄金对 A股 / 港股的传导影响
- 每个核心市场尽量给出判断、逻辑、风险点和区间
- 避免单边拍脑袋式预测，必须做条件推演
- 结论必须能落地到仓位管理和方向选择
- 最后附上“以上仅供参考，不构成投资建议”

## 我建议你以后固定追加的指令

如果你希望输出更稳定，建议每次在提示词最后再附加以下约束：

1. “如果数据存在滞后，请先声明数据时点，再展开分析。”
2. “不要只复述新闻，要解释新闻如何通过原油、美元、黄金、美股、A股、港股进行传导。”
3. “不要只判断指数，还要判断行业主线、资金风格和高低切换。”
4. “如果市场处于关键事件窗口，必须以情景推演代替绝对预测。”
5. “若用于 A股 / 港股盘前，请把昨晚美股走势作为前置章节，不可省略。”

---

## 一句话总结

以后写盘前或周度市场分析，最核心的不是堆信息，而是先抓住市场总开关，再用跨资产联动和情景推演把判断落到节奏、结构、仓位和配置上。
