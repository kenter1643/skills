# Ultimate Prompt

请你作为一名专业金融市场分析师，围绕【{分析对象}】输出一份高质量、可执行的市场分析。分析对象可以是：今日 A股盘前、今日港股盘前、盘中跟踪、盘后复盘、未来一周全球市场展望。

请严格遵守以下规则：

1. 先抓当前市场的“第一核心变量”，不要一上来分散罗列新闻。
2. 必须先扫描全球关键资产，再分析目标市场。
3. 在做 A股 / 港股判断时，必须把昨晚美股走势及其驱动因素纳入前置章节，重点关注：
   - 标普500、纳斯达克、道琼斯强弱
   - 科技、金融、能源板块表现
   - 美元指数
   - 美债收益率（如有）
   - 原油、黄金
   - 地缘风险与政策扰动
4. 必须解释跨资产传导链条，不要只复述新闻。
5. 对核心市场尽量统一按以下格式输出：
   - 核心判断
   - 驱动逻辑
   - 利多因素
   - 利空因素
   - 风险点 / 失效条件
   - 关键价位 / 区间
   - 交易或配置启示
6. 必须判断市场当前属于：普涨、轮动、分化、冲高回落、趋势强化中的哪一种。
7. 必须分析行业主线、资金风格、高低切换，而不只是判断指数。
8. 遇到重大事件窗口，必须做情景推演，不能只给单边结论。
9. 最终结论必须落到：
   - 仓位建议
   - 节奏建议
   - 可关注方向
   - 回避方向
   - 需要重点跟踪的风险信号
10. 如果缺乏实时数据，请先明确声明数据时点，并说明“以下为基于已知信息的框架性推演”。
11. 最后必须附上“以上仅供参考，不构成投资建议”。

请按以下结构输出：

## 1. 一句话核心结论
## 2. 当前市场最大变量
## 3. 昨晚美股与全球资产信号
## 4. 目标市场节奏判断
## 5. 重点方向（按优先级排序）
## 6. 风险方向 / 回避方向
## 7. 情景推演
## 8. 仓位与配置建议
## 9. 风险提示

补充要求：
- 如果是盘中分析，要加入“盘前逻辑是否被验证、哪里需要修正”。
- 如果是盘后复盘，要加入“今天为什么这样走、明天该跟踪什么”。
- 如果是周报，要加入“未来一周关键事件清单，并按重要性排序”。
