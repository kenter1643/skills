# financial-market-analysis

一个面向 A股、港股与全球市场的金融分析 skill，覆盖：
- 盘前分析
- 盘中跟踪
- 盘后复盘
- 周度展望

核心方法不是简单堆新闻，而是：
1. 先抓主导市场的核心变量
2. 再做全球资产扫描
3. 用跨资产传导解释市场演绎
4. 最后落到节奏、结构、仓位与风险控制

## 文件结构

- `SKILL.md`：主技能说明
- `references/prompt-template.md`：通用提示词与长模板
- `references/thinking-summary.md`：方法论总结
- `references/example-output.md`：示例输出
- `references/ultimate-prompt.md`：终极一键复用 prompt
- `templates/premarket.md`：盘前模板
- `templates/intraday.md`：盘中模板
- `templates/postmarket.md`：盘后复盘模板
- `templates/weekly-outlook.md`：周报模板

## 适用场景

- A股 / 港股盘前判断
- 盘中判断主线是否强化、轮动还是分歧扩大
- 收盘后做结构化复盘
- 对未来一周做多资产联动展望
- 将任意金融分析内容提炼为可复用框架

## 关键特点

- 强制纳入昨晚美股走势及其驱动因素
- 强制观察科技 / 金融 / 能源板块
- 强制分析美元、人民币、原油、黄金对 A股 / 港股的传导
- 强制给出风险点、关键位、区间与仓位建议
- 强制做情景推演，避免拍脑袋式单边预测
