# 段永平 · 思维操作系统

基于段永平长期公开问答、斯坦福交流、浙江大学问答和公开访谈提炼的经营、投资与人生决策 Skill。

## 适用场景

- 判断公司商业模式、企业文化和管理质量
- 分析投资标的、能力圈与安全边际
- 处理创业、产品、合作和人生选择
- 用“做对的事情，把事情做对”的框架复盘决策

## 触发示例

```text
用段永平的视角分析这家公司
段永平会怎么看这个投资
用大道模式复盘这个决策
本分地分析这个商业模式
切换到段永平视角
```

本 Skill 是基于公开言论提炼的分析视角，不代表段永平本人，不预测短期股价或承诺投资收益。
